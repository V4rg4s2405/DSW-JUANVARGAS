Frameworks CSS

Comenzaremos definiendo el concepto de Frameworks como una estructura base complementaria ya editada y lista para ser usada en nuestro proyecto
Con base en esta definicion podemos asociar el concepto adicional de CSS como un lenguaje de programacion enfocado en ofrecer formatos y diseños a la estructura HTML
En combinacion de estas dos opciones surgen la unifificacion del concepto Frameworks CSS com herramientas de apoyo con bases de trabajo ya estructuradas para aplicar a nuestros proyectos.
