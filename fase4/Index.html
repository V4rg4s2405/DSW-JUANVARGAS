Prposito del sitio

Este sitio busca suplir la necesidad de adquirir formas educativas del lenguaje de prorgmacion HTMl mediante la elaboracion de objetos visuales para las tematicas Fundamentos de CSS, Frameworks CSS

Introduccion Diseños Web

El curso diseños web toma como base para su aprendizaje la solucion de un proyecto mediante lecturas de las tematicas de los lenguajes de programacion HTML y su complemento de diseño Css.

El primero cuenta con una seire de etiquetas las cuales modelan toda la estructura de lo que hoy conocemos como paginas web es el esqueleto las bases de la pagina

Para el segundo tema (Css) su esencia esta en ofrecer vitalidad al sitio, ofreciendole diseños de orientaacion, color, tamaños y tipos de letra.

En este orden de ideas se busca dar a conocer los lenguajes mediante el uso en un proyecto mediante tareas anidadas en el desarollo del curso, en el cual vamos ir avanzando paso a paso creando un repositorio, luego una estructura base, para añadir luego el complemento de siseño con css, para esta oportunidad tengo la oportunidad de trabajar los temas Fundamentos Css, Frameworks CSS
