Fundamentos de CSS

CSS ó Acronimo de Hojas de estilo en cascada o cascading style sheet es tambien al igual que su herramienta amiga de trabajo HTML un lenguaje de programacion permite aplicar diferentes estilos a los diseños Web de forma que se pueda tener una simetria en los diseños en tanto colores como tipo de letras. Como todo lenguajede programacion este se rige bajo normas y parametos en su estructura y sintaxis de desarrollo, siendo esta la forma en ue se compone:

  • Selector: Indica que etiqueta para Html se trabajara con el estilo
  • Propiedad: indica que propiedad de la etiqueta se vera afectada, como estilo de letra color o de fondo
  • Valor: valor de la propiedad, como puede ser tamaño de la letra o ancho de una marger

Cuentan con la propiedad de definir varias propiedades en una solo selector. Ej Selector(tamaño, color de fondo) al ser un lenguaje independiente, pero totalmente complementario con HTML, CSS se puede declarar de varias maneras.

  • Aunque no es la mas recomendada, se puede usar la sintaxis para CSS en combinacion con el codigo de HTML, mas no seria lo mas ideal ya que al momento de validar una posible inconsistencia las lineas de codigo serian mas extensas y tener una mayor difiicultad para diferenciar la sintasix de uno y otro lenguaje
  • La forma mas recomendada es mediante la separacion del codigo, en un apartado diferente a nuestro codigo HTML, bastara con apuntar los estilos, y formas a las etiquetas o objetos que contenga nuestro codigo HTML
